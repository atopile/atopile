# faebryk

Open-source software-defined EDA

## Get running
```
> git clone git@github.com:faebryk/faebryk.git
```
```
> cd faebryk
```
```
> git checkout feature/add_sexp_generator
```
```
> git submodule init
```
```
> git submodule update
```
```
> ./source/main.py
```
